# 股票信息系统 - 前端设计稿

## 1. 项目概述

- **项目名称**: Stock Info System (股票信息收集系统)
- **项目类型**: Web 管理系统
- **核心功能**: 股票行情查询、自选股管理、新闻资讯、用户管理
- **目标用户**: 个人投资者

## 2. 技术选型

- **方案**: HTML5 + Bootstrap 5 + jQuery
- **图表库**: ECharts (股票K线/走势)
- **优点**: 轻量级、无需构建工具、快速开发

## 3. 页面结构

### 3.1 页面列表

| 页面 | 路由 | 功能 |
|------|------|------|
| 登录页 | `/login` | 用户登录 |
| 注册页 | `/register` | 用户注册 |
| 首页/仪表盘 | `/` | 概览、自选股行情 |
| 股票行情 | `/stocks` | 行情列表、搜索 |
| 股票详情 | `/stock/<symbol>` | K线、新闻、公告 |
| 自选股管理 | `/watchlist` | 自选股 CRUD |
| 热门股票 | `/hot` | 热门股票排行 |
| 个人设置 | `/settings` | 推送设置 |

## 7. 响应式设计

| 断点 | 屏幕宽度 | 布局变化 |
|------|----------|----------|
| Mobile | < 576px | 单列，隐藏侧边栏 |
| Tablet | 576px - 992px | 两列，侧边栏折叠 |
| Desktop | > 992px | 完整布局 |

## 8. API 接口对应

| 前端页面 | API 接口 |
|----------|----------|
| 登录 | POST /api/v1/user/login |
| 注册 | POST /api/v1/user/register |
| 首页数据 | GET /api/v1/user/watchlist |
| 股票列表 | GET /api/v1/stock?limit=50 |
| 股票详情 | GET /api/v1/stock/{symbol}/realtime |
| 热门股票 | GET /api/v1/stock/hot |
| 公告 | GET /api/v1/stock/{symbol}/announcements |
| 添加自选 | POST /api/v1/user/watchlist |
| 删除自选 | DELETE /api/v1/user/watchlist/{symbol} |
| 推送设置 | POST /api/v1/user/preference |
